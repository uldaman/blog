(四) 安装 Eclipse

Thu 11 June 2015

Eclipse 是开发 Java 的 IDE (集成开发环境).

可以直接在官网上下载最新版本的 Eclipse, 不过它是英文版的, 我们需要手动下载语言包, Eclipse有一个语言包项目, 叫Eclipse Babel Project, 如果需要语言包, 可以联机从这儿下载. http://www.eclipse.org/babel/

找到下载页面后, 还要根据自己使用的 Eclipse 的版本来确定在哪下载.

可以看到我的是 Luna 版本.

点击第一张图上的 Luna, 跳转到语言包下载界面, 找到中文包, 下载, 解压, 覆盖即可.

再次打开 Eclipse 就是中文的了. 嬜
