title: 正则表达式

python 中支持正则表达式的模块是 re, 使用时 import re 即可.

**re 常用命令:**
findall: 匹配所有符合规律的内容, 返回包含结果的列表.
search: 匹配第一个符合规律的内容, 返回一个正则表达式对象.
sub: 替换符合规律的内容, 返回替换后的值.
