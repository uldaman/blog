《Python 开发实战》书评

Par @Martin dans [ 读书笔记 ] le Fri 05 August 2016
Tags :

有种书只看目录就很有收获, 但看内容, 收获却增加不了多少, 我想这大概说的就是这本书…

该书介绍了使用 Python 开发一个项目所需的环境配置、流程、工具、测试 (除个别外, 其它的都不局限于 Python), 所以这本书很适合做为 UP 中项目管理和环境科目的参考书.

另外书中对介绍的知识浅尝辄止, 需要再搜索书中关键字, 找点补充读物进行学习.
