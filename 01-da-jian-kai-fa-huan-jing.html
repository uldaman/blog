01、搭建开发环境

1. golang 安装

地址: https://golang.org/dl/, 去下载合适的版本, 直接安装即可, 安装完成后, 检查下 bin 目录是否被添加到系统环境变量了.

2. 开发工具安装

开发工具可以选择 LiteIDE 或者 安装了 GoClipse 插件的 Eclipse.

这里我选择的是 LiteIDE, 下载地址: http://sourceforge.net/projects/liteide/

然后还要安装个 gcc, 下载地址: http://www.pc6.com/softview/SoftView_110479.html, 也可以直接搜索 gcc for windows 自行下载.

解压 LiteIDE 压缩包, 运行 bin 目录下的 liteide.exe.

设置当前系统

我这里选择的是 win 64.

设置 go 环境

查看->编辑当前环境, 注意红框里的内容.

设置 GOPATH

查看->管理GOPATH, 创建新目录 G:\GoProject, 并添加到 “自定义目录”.

3. 测试

文件->新建

选择上一步添加的 GOPATH, 模板为 Go1 Command Project

然后, LiteIDE 会自动给你生成个 main.io 文件, 内容如下:

// test project main.go
package main

import (
    "fmt"
)

func main() {
    fmt.Println("Hello World!")
}

编译->编译配置

在 BUILDARGS 添加 -gcflags “-N -l”, 目的是去掉编译优化, 方便调试.

开始调试

点击按钮 BR, 或者 “FR”

按键盘 F9 可设置断点.
